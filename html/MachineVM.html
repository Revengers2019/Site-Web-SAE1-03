Machine Virtuelle

Une machine virtuelle (VM) est un environnement simulé qui fonctionne à l'intérieur d'un ordinateur physique (appelé hôte). Elle opère comme un ordinateur indépendant, avec son propre système d'exploitation (OS) et ses applications, tout en étant séparée des autres machines virtuelles et du système de l'hôte.

Exemple :

  • Développement logiciel : Un développeur crée une VM avec Linux pour tester son application sans affecter son PC principal sous Windows.
  • Simulation de réseau : Une entreprise utilise des VMs pour simuler un réseau complexe et tester la performance et la sécurité des systèmes.

Ce que nous avons fait :

  1. Créé une machine virtuelle.
  2. Configuré les ressources (mémoire, disque) pour optimiser les performances de la machine virtuelle.
  3. Installé un système d'exploitation et configuré Apache pour héberger notre site web.
  4. Hébergé la machine virtuelle créée sur un serveur Apache, puis configuré pour rediriger les requêtes vers la machine virtuelle.
  5. Installé notre site web dans notre machine virtuelle créée et configurée.
